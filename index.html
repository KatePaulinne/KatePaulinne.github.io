Cave Maze

Unity3D Game made in 6 weeks.

Cave Maze is the first project I have made in Unity 3D. The game mechanics were inspired by ‘Weeping Angels’ from Doctor Who, as I wanted to come up with an idea that would be something different from what was already made. Despite spending a lot of time working on this project, I can still see a lot of features that could be implemented. Overall I got good feedback on my first game both at University and after publishing it on Kongregate.

  • Date: March 2014
  • Project made for: AINT102 Games Workshop module
  • Category: Games Development

Alien Game

Made using XNA Framework.

Alien Game is a project made with XNA Framework in Microsoft Visual Studio. Working with my course mate - Sam Eden, helped me improve teamwork and communication skills. As well as this it gave me an understanding of writing a game without using any interface, just pure coding. Instead of using online resources we have created our own graphics for this game.

  • Date: April 2014
  • Project made for: SOFT144 Software Engineering for Games Development module
  • Category: Games Development
